Installing Quadify

Quadify • Main software installation

What you’re about to install

Quadify is installed as a Volumio plugin.

This means the installation process, updates, and requirements are maintained on GitHub and may evolve over time.

For that reason, this guide does not duplicate the install steps. Instead, it shows you how to follow the official instructions correctly.

Before you continue

You should already have:

  • Volumio installed and running
  • Network access working
  • SSH enabled
  • A terminal connection open (PuTTY or Termius)

If you haven’t done this yet, go back to the Headless Setup & Terminal Access step.

Go to the Quadify GitHub page

Open the Quadify GitHub repository in your browser:

https://github.com/theshepherdmatt/Quadify-Plugin

Take a moment to read the page. In particular, pay attention to:

  • The project description at the top
  • The Installation section
  • Any notes or warnings in the README

Follow the installation steps

With your terminal already connected to the Raspberry Pi, follow the installation steps exactly as written on the GitHub page.

Copy and paste commands carefully. If something changes in the future, GitHub will always reflect the correct method.

When the install completes

Once the plugin is installed:

  • Reboot the Raspberry Pi if instructed
  • Open the Volumio web interface
  • Check that Quadify appears in the Plugins section

From this point on, Quadify is managed entirely through Volumio.
