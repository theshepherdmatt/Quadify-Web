Quadify Setup Guide

This guide walks you through installing, wiring, and configuring Quadify on a Raspberry Pi running Volumio. Follow the steps in order — each one builds on the previous.

Recommended setup order

  1. Install Volumio
    Prepare the SD card and flash Volumio.
  2. Connect Volumio to Wi-Fi
    First boot, network setup, and DAC configuration.
  3. Headless Setup & Terminal Access
    Enable SSH and connect to the Pi remotely.
  4. Install Quadify (Plugin)
    Follow the official GitHub instructions.
  5. Wiring the Modules
    Power, screen, rotary encoder, and terminal HAT.
  6. OLED Screen Setup
    Check and wire the SSD1322 display correctly.
  7. MCP23017 Board Install
    Expand GPIO for buttons and LEDs.
  8. Buttons & LEDs
    Wire the button matrix and LED outputs.

A few notes before you start

  • Take your time — neat wiring saves hours later
  • Double-check power pins before applying power
  • Follow the steps in order
  • If something doesn’t work, don’t panic — it’s usually wiring

Quadify is designed to be simple, robust, and serviceable.
